# school-api

## Setup 
1- Certifique-se de ter feito as seguintes instalações:
- node 20.19.6
- docker

2- Com o docker aberto, monte as imagens:
-  docker compose up -d

3- Instale as dependências do projeto:
- yarn install

4- Rode as migrações:
- yarn run:migrations

4- Execute o servidor:
- yarn start
